# LedControlPanel

Устройство __LedControlPanel__ предназначено для регулировки цветовой температуры и яркости светодиодной ленты.

  <img src="https://github.com/ilyaadilov/LedControlPanel/blob/main/Docs/FrontDevicePicture.png" alt="Изображение">
</div>

Регулировка осуществляется вращением ручек, включение/выключение освещения – кнопкой.
Устанавливается под полку, к устройству подключаются две светодиодные ленты разных цветовых температур.
Левая ручка отвечает за изменение цветовой температуры посредством увеличения или уменьшения одной из двух светодиодных лент.
Правая ручка отвечает за яркость обеих лент.

__Основные характеристики:__

- входное напряжение DC – 24 В;
- максимальный постоянный ток Led ленты – 5 А;
- частота ШИМ = 25 кГц;
- диапазон регулировки яркости – 0-100% с шагом 1%;
- диапазон регулировки цветовой температуры – в зависимости от подключаемых лент (например, 3000К и 6000К, тогда диапазон 3000-6000К с шагом 1%).
